## Introduction
This project is a simple React application that includes user authentication and a dashboard. It features a home page, register form, login form, and a dashboard. The project uses context for authentication and admin management.

## Features
- User registration
- User login
- Dashboard
- Responsive design

## Technologies Used
- React
- React Router
- Context API
- CSS for styling

## Setup and Installation
Follow these steps to set up and run the project on your local machine:

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-repo.git
    cd your-repo
    ```

2. Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

## Running the Project
To start the development server, run:
```bash
npm start
# or
yarn start
